# 사내 출퇴근 시스템

### 📌 Project Overview  
JPA공부를 위한 간단한 사내 출퇴근 시스템 

---

## 🚀 Features
- **팀 관리**: 팀 등록 및 팀 조회.  
- **직원 관리**: 직원 추가 및 조회.  
- **출퇴근 관리**: 직원들 출퇴근 기능 및 출퇴근 기록 조회.  
- **연차 관리**: 직원들 연차 사용.  

---

## 🛠️ Tech Stack
- **Language**: Java 17  
- **Framework**: Spring Boot  
- **ORM**: JPA (Hibernate)  
- **Database**: MySql  
- **Build Tool**: Gradle
